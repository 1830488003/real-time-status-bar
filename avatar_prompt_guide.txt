### **<模块化功能：AI动态头像指南>**

**核心指令：** 当本指南被激活时，你必须为状态栏中的每个角色生成一个头像。

---

### **第一步：学习HTML结构**

你**必须**使用以下HTML结构来展示头像和角色名称。将它插入到 `.content-pane` 元素的顶部。

```html
<div class="character-header">
    <div class="character-avatar">
        <img src="[此处替换为选择的URL]" alt="[角色名称]">
    </div>
    <h2 class="character-name">[角色名称]</h2>
</div>
```

---

### **第二步：学习CSS样式**

你**必须**将以下CSS代码添加到主 `<style>` 标签内，以确保头像和名称的样式正确。

```css
/* --- 头像模块专属样式 --- */
.character-header {
    display: flex;
    align-items: center;
    gap: 15px;
    margin-bottom: 15px;
}
.character-avatar {
    width: 60px;
    height: 60px;
    border-radius: 50%;
    overflow: hidden; /* 确保图片被裁剪为圆形 */
}
.character-avatar img {
    width: 100%;
    height: 100%;
    object-fit: cover; /* 确保图片不变形地填满容器 */
    border: 2px solid rgba(132, 102, 218, 0.7);
    border-radius: 50%;
}
/* 移动端适配 */
@media (max-width: 600px) {
    .character-header {
        gap: 10px;
    }
    .character-avatar {
        width: 50px;
        height: 50px;
    }
}
/* --- 样式结束 --- */
```

---

### **第三步：选择并使用URL**

你将会收到一个“可用头像URL列表”。在构建角色信息时，你**必须**：
1.  仔细阅读每个URL旁边的描述。
2.  根据当前角色的性别、气质和情景，从列表中选择一个**最贴切**的URL。
3.  将选中的URL放入 `<img>` 标签的 `src` 属性中。
